for version 0.2.4 I would to perform some organizational changes to the UI:

* Switch to a 2 column layout
    * Left column will contain 2 divisions
        * Top division will contain tabs for:
            * Projects
            * Favorites - with it's own sub-tabs:
                * Project Favorites
                * Global Favorites
            * Files - with it's own sub-tabs:
                * Checked Out Files
                * Reference Files
            * Notes
        * Bottom division will contain the same items as before:
            * Directories section with tabs for:
                * Source (Tracked Directories/Directory Browser)
                * Local (Tracked Directories/Directory Browser)
    * Right column will contain the expanded Files section
        * The current source folder will now be shown here instead of in the directory browser
        * Extension filter becomes a button that should bring up a dialog window for filter controls. A short preview of the current settings will be shown in an adjacent label.
        * The files search box sits adjacent to the Extension filter/label aligned to the right of the screen.
        * The files list takes the rest of the available space
        * The previous section for "Directory - Controlled Files" and "Directory - Notes" will move to it's own pop-up window that can be activated/located through the Files menu (rather than taking up screen real-estate)
